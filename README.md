# SymphonyAI Webflow Code Components

This repository contains reusable Webflow code components used for the SymphonyAI project.

## Development

Run the project locally to test components:

```bash
npm run dev
```

## Create a Component

Generate a new component:

```bash
npm run create
```

## Share Components

Share the components to Webflow:

```bash
npx webflow library share
```
